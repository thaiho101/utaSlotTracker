# UTA Slot Tracker – Parking Availability System

## 🚀 Overview
A front-end web application developed as part of a team project to improve campus parking efficiency by displaying real-time parking availability across multiple locations.

## 🔥 Key Features
- Visual map-based interface for parking locations
- Real-time display of available slots
- Interactive UI for quick navigation
- Focus on usability and responsive design

## 📊 Impact
- Improved user awareness of parking availability
- Reduced time spent searching for parking on campus

## 🛠 Tech Stack
- JavaScript
- HTML
- CSS
